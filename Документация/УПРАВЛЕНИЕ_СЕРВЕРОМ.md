# Управление сервером KopiBar

## Подключение к серверу

Открой PuTTY, введи:
- Host: `77.239.105.144`
- Port: `22`
- Нажми Open → логин `root` → пароль

---

## Основные команды PM2

### Статус — посмотреть работает ли сервер
```bash
pm2 status
```
Если в колонке status написано **online** — всё хорошо.
Если **stopped** или **errored** — нужно перезапустить.

### Запустить сервер
```bash
pm2 start /root/kopibar-server/server.js --name kopibar
```

### Остановить сервер
```bash
pm2 stop kopibar
```

### Перезапустить сервер (например после изменения server.js)
```bash
pm2 restart kopibar
```

### Удалить из PM2 полностью
```bash
pm2 delete kopibar
```

### Логи — посмотреть что происходит внутри
```bash
pm2 logs kopibar
```
Выйти из логов: `Ctrl + C`

### Последние 100 строк логов
```bash
pm2 logs kopibar --lines 100
```

---

## Обновление server.js

Если нужно заменить файл сервера:

1. Загрузи новый `server.js` через WinSCP в `/root/kopibar-server/`
2. В PuTTY выполни:
```bash
pm2 restart kopibar
```

---

## Проверка что сервер отвечает

Открой в браузере:
```
http://77.239.105.144:3001/ping
```
Ответ должен быть: `{"ok":true,...}`

Посмотреть сколько данных в кэше:
```
http://77.239.105.144:3001/cache-status
```

---

## Перезагрузка самого VPS

Если сервер завис полностью — зайди в панель U1host:
`https://vm.u1host.com`

Найди сервер → кнопка **Перезагрузить**.

После перезагрузки PM2 запустит server.js автоматически (это настроено командой `pm2 startup` + `pm2 save`).

## Потребление ресурсов
```bash
pm2 monit
```
Показывает CPU и RAM в реальном времени. Выйти: `Ctrl + C`

## Просмотр файлов на сервере

Перейти в папку сервера:
```bash
cd /root/kopibar-server
```

Посмотреть файлы:
```bash
ls -la
```

Посмотреть содержимое server.js:
```bash
cat server.js
```

---

## Если что-то сломалось

**Шаг 1** — посмотреть логи:
```bash
pm2 logs kopibar --lines 50
```

**Шаг 2** — перезапустить:
```bash
pm2 restart kopibar
```

**Шаг 3** — если не помогло, запустить вручную для диагностики:
```bash
pm2 stop kopibar
cd /root/kopibar-server
node server.js
```
Ошибка появится прямо в терминале. Выйти: `Ctrl + C`, потом снова `pm2 start kopibar`

**Шаг 4** — проверить свободную память:
```bash
free -h
```

**Шаг 5** — проверить открыт ли порт:
```bash
ufw status
```
Должна быть строка `3001   ALLOW`
